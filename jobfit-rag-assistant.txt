# JobFit RAG Assistant

## Project Overview

JobFit RAG Assistant is an AI-powered system that helps job seekers analyze job descriptions against their resumes and project portfolios.

The system will:

* Analyze Job Descriptions (JDs)
* Match resume skills with job requirements
* Identify skill gaps
* Summarize job descriptions
* Answer questions about job postings
* Recommend relevant projects to showcase
* Retrieve supporting evidence using RAG

This project is being built incrementally while learning NLP, LLMs, Vector Databases, and Retrieval-Augmented Generation (RAG).

├── requirements.txt
├── .gitignore

### Day 1 — Similarity Engine

Goal: Create the core matching module using Sentence-BERT.

Tasks:
- Create `src/jobfit/` package
- Create `similarity.py`:
  - Load Sentence-BERT (`all-MiniLM-L6-v2`)
  - Encode resume bullets + JD requirements
  - Compute cosine similarity matrix
  - Print top-K matching resume bullets per JD requirement
- Test with existing `data/resume_sentences.txt` and `data/jd_sentences.txt`

Deliverable:
- `src/jobfit/__init__.py`
- `src/jobfit/similarity.py`
- Working similarity output

Status: **In Progress**

---

### Day 2 — JD Summarizer + QA Pipeline

Goal: Add HF pipeline modules for summarization and question answering.

Tasks:
- Create `summarizer.py` with `summarize_jd(jd_text) -> str`
- Create `qa.py` with `answer_question(jd_text, question) -> str`
- Test both independently

Deliverable:
- `src/jobfit/summarizer.py`
- `src/jobfit/qa.py`

Status: **Planned**

---

### Day 3 — Orchestrator (analyzer.py)

Goal: Wire everything into a single `analyze()` function.

Tasks:
- Create `analyzer.py`
- `analyze(resume_path, jd_path)` loads data, runs similarity → summarizer → QA
- Prints a clean report

Deliverable:
- `src/jobfit/analyzer.py`
- End-to-end working output

Status: **Planned**

---

### Day 4 — Polish + Edge Cases

Goal: Handle real-world inputs gracefully.

Tasks:
- Handle empty files, long JDs, multiple questions
- Add `__main__.py` so `python -m jobfit` works
- Test with a real JD from LinkedIn

Deliverable:
- `src/jobfit/__main__.py`
- Robust CLI entry point

Status: **Planned**

---

### Day 5 — README + Final Verification

Goal: Document + verify everything.

Tasks:
- Update `README.md` with usage and example output
- Create `requirements.txt` with pinned versions
- Run final end-to-end test

Deliverable:
